Score of Parentheses

The Question

Given a balanced parentheses string s, return the score of the string. The score of a balanced parentheses string is based on the following rule:
  1. "()" has score 1.
  2. AB has score A + B, where A and B are balanced parentheses strings.
  3. (A) has score 2 * A, where A is a balanced parentheses string.

My Thoughts

At first glance, this problem looks like it can be solved by recursion. Since everything inside a set of parentheses are multiplied by two, each parentheses can be thought of like a smaller subproblem with () as the base case.

However, I think there is a better way to solve this problem, without recursion. If I keep track of the "depth" of the current parentheses, I can use distribution to calculate the score instead. Thus, "(" will be my sign to increase the depth, and ")" will be my sign to decrease depth. Furthermore, I only increase the score if there is a pair of empty parentheses. I can do this by checking the last character of the list.

Next comes the issue of storing the depth. I can see that the score will result with 2^x, x being the depth, instead, I can just store the depth as this value and avoid calculating it in real time. I will start the depth as 0.5, so increasing it by a factor if two will give me 1.

Basically, my code will be as follows:

  1. Declare total, and depth
  2. Make a look that iterates through the input

    • If I see "(", increase the depth
    • If I see ")", check if it's an empty parentheses (add to total if it is) and decrease depth
  4. Return total

And here it is with actual code.

def scoreOfParentheses(self, s: str) -> int: depth = 0.5 total = 0 for i in range(len(s)): if s[i] == "(": depth = depth * 2 elif s[i - 1] == "(": total = total + depth depth = depth / 2 else: depth = depth / 2 return int(total)
