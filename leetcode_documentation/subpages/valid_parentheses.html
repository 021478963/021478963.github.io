The Question

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.
An input string is valid if:
  1. Open brackets must be closed by the same type of brackets.
  2. Open brackets must be closed in the correct order.
  3. Every close bracket has a corresponding open bracket of the same type.

My Thoughts

My first question after reading this question was, "What does in order mean?" I guess it could mean that a parentheses opened before another one must also be closed before the other one. I'm going to test my hypothesis with "[]([)]," which I expect to output false, and "{([])}," which I expect to output true.

My thoughts were correct. That means, when I see a closing bracket, I need to figure out what the closest opening bracket was, and compare it to that. One type of data structure jumps out to me --a stack. The last in, first out nature of a stack would allow for me to push all opening brackets onto the stack as I see them, and I would be able to retrieve them as needed when I see a closing bracket.

I'm going to use a for loop iterating through every element in the list, with the current element as my variable. The thought of revurion popped into my head initially, but I don't think the extra variables stored on the call stack would be worth the simplicity.

For every element, I will first check if it's a closing bracket. If not, I just push it onto the stack. If it is, however, I need to get its type, and compare it to the last opening bracket. If it doesn't match, I know I can instantly return false. Otherwise, I will move on and check all other elements in the list.

In essence, my code is as follows:

  1. I make a stack to hold all of my brackets
  2. I make a loop to iterate through all elements of the input
  3. I check all the elements one by one. If it's an opening bracket, I push it onto the stack. Otherwise, I will do comparison.
def valid_parentheses(): previous_brackets = [] for i in s: if i == '(' or i == '[' or i == '{': previous_brackets.push(i) elif i == ')': if previous_brackets.pop() != '(': return False elif i == ']': if previous_brackets.pop() != '[': return False else: if previous_brackets.pop() != '{': return False return True

I'm not sure why, but I assumed that the input would be a valid pair of parentheses. Once I check for that, the code above works.
